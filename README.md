#Medical Store Management System

##Overview
Medical Store Management System is a java based application designed to manage daily operation of Medical Stores.It helps medicine inventory,supplier details,customer records,billing and sales reports.

##Features
-Medicine Management 
-Inventory Management
-Customer Management
-Supplier Management
-Billing System
-Sales Reports
-Expiry Date Tracking
-Low Stock Alert
-User Authentication

##Technologies Used
-Java
-Maven
-JDBC
-MySQL
-Git & GitHub
-IntelliJ IDEA
